# Library Management System
A Library Management System made with Python & PyQt5

Library System :
    - add new book
    - aditing book
    - deleting book
    - catagories
    - search
    - users , login , signup
    - settings [catagories , author , publisher]
    - day to day operations
    - reports [exel files]

    - desc
    - catagory
    - price

    - publisher

users :
